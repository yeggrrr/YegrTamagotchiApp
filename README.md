# 🌱 다마고치 앱 만들기 🌱
#
# 📌 앱 설명
### 다마고치를 선택하고, 물과 밥을 주며 다마고치를 키우는 앱입니다.
#
## 📌 구성
- CodeBase
- 선택 화면 (다마고치 종류 선택)
- 상세 화면 (다마고치 소개 & 취소 및 시작 버튼)
- 메인 화면 (다마고치 키우기 - 물주기, 밥주기)
- 설정 화면 (닉네임 변경, 다마고치 변경, 데이터 초기화)
- 이름 변경 화면
- 다마고치 변경 화면 (상세 팝업 - 시작하기 버튼 -> 변경하기 버튼)
- 데이터 초기화 화면
